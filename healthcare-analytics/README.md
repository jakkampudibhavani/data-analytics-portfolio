## CMS Hospital Readmissions Analysis (HRRP)

### Overview
This project analyzes CMS Hospital Readmissions Reduction Program (HRRP) data to identify
high-risk conditions, state-level performance trends, and hospitals exceeding CMS benchmarks.

### Tools Used
- SQL Server
- Python (Pandas)
- Power BI
- Excel

### Key Analyses
- Condition-level excess readmission comparison
- State-level performance vs CMS benchmark
- Impact of low discharge volume on readmission metrics

### Key Insights
- Hip/Knee replacement and COPD show higher excess readmission ratios
- Several states consistently exceed CMS benchmarks
- Low-volume hospitals exhibit higher variability
